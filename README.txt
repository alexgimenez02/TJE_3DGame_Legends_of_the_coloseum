Como jugar:
	-> Menus:
		-> Click izquierdo para seleccionar la opcion que cambie al pasar por encima
	-> Gameplay
		-> W para mover hacia adelante
		-> A para mover hacia la izquierda
		-> S para mover hacia atras
		-> D para mover hacia la derecha
		-> Esc para abrir el menu
		-> Click izquierdo para interactuar o atacar
		-> '→' Defensa derecha
		-> '←' Defensa izquierda
		-> '↑' Defensa arriba
		-> Mover mouse hacia izquierda o derecha para mover la camara

IMPORTANTE:
No hacer resize ya que causará bugs incontrolables y puede llegar a causar algun crash que otro.
Una vez se llega al máximo de estadísticas, cuando se interactua con el barman, se llega a un softlock.
Intentar mantenerse en la zona de juego, ya que hay zonas donde las colisiones te permiten "salir de las barreras".
Hay un bug, que causa que si guardas partida en la Arena, si sales y vuelves a entrar una primera vez, el combate se cambiará, intentar no replicar.
Algunas veces los objetivos no aparecen en la zona de juego y causa confusión.
La musica lleva copyright, asi que mejor no publicar en ningún sitio sin licencias
